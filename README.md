# Snake Game 🐍

This is a simple Snake game built using Python and Pygame.

## 🎮 Features
- Snake movement using arrow keys
- Food spawning system
- Score system
- Game over when hitting walls or yourself

## 🛠️ Technologies Used
- Python
- Pygame

## 📚 What I Learned
During this project, I learned:
- Basic game development concepts
- How to work with coordinates (x, y)
- Event handling (keyboard input)
- Lists and logic for snake movement
- Collision detection

## 🧠 Development Process
This project took me around **2 days** to understand and build.

I used **vibe coding** (with ChatGPT assistance) to help me structure and understand parts of the code, but I made sure to understand how everything works step by step.
